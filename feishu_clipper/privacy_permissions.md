# 隐私权政策（飞叔转存大师）

最后更新日期：2026-06-01

本隐私权政策适用于浏览器扩展程序「飞叔转存大师」（英文名：Cloud Clip Master，以下简称“本扩展”）。本扩展用于将用户当前浏览的网页正文、标题、图片等内容提取并保存到用户授权登录的飞书/Lark 云文档或知识库中。

## 1. 单一用途说明

本扩展的单一用途是：在用户主动触发剪存操作时，提取当前网页中适合阅读和归档的正文内容，并通过飞书/Lark 服务保存为云文档或知识库内容，帮助用户进行网页资料整理、转存和归档。

本扩展不会用于广告投放、跨站追踪、用户画像、信用评估或与网页剪存无关的用途。

## 2. 权限使用说明

### 2.1 activeTab 权限

**请求理由**：用于在用户点击扩展图标、右键菜单或页面悬浮按钮时，访问当前活动标签页的信息和内容，以执行网页正文提取、剪存和页面交互操作。该权限仅在用户主动触发扩展功能时使用。

### 2.2 tabs 权限

**请求理由**：用于获取当前标签页的 URL、标题和标签页 ID，并在必要时向对应标签页发送消息，例如打开剪存面板、显示/隐藏页面内悬浮按钮、检测当前页面是否支持剪存等。

### 2.3 contextMenus 权限

**请求理由**：用于在浏览器右键菜单中提供剪存入口、语言切换、自动打开文档、悬浮按钮开关等扩展功能选项。用户可以通过右键菜单主动触发相关操作。

### 2.4 cookies 权限

**请求理由**：用于识别用户在飞书/Lark 服务中的登录状态，并在用户授权登录后调用飞书/Lark 官方接口完成文档创建、内容上传、图片上传等剪存流程。本扩展不会读取与剪存服务无关的网站 Cookie，也不会将 Cookie 提供给扩展作者或其他第三方。

### 2.5 storage 权限

**请求理由**：用于在浏览器本地保存用户偏好设置和必要状态，包括：

- 当前语言设置
- 是否自动打开剪存后的云文档
- 是否显示剪存悬浮按钮
- 悬浮按钮位置
- 用户登录状态缓存
- 剪存配置、站点规则和功能开关

这些数据主要存储在 `chrome.storage.local` 和页面 `localStorage` 中，用于维持扩展功能和用户偏好。

### 2.6 scripting 权限

**请求理由**：用于在用户访问网页或主动触发剪存时，向页面注入必要脚本，以提取网页正文、处理图片、显示剪存悬浮按钮、打开剪存确认界面或执行可读性解析。脚本仅用于实现网页剪存功能。

### 2.7 主机权限（Host Permissions）

本扩展请求 `*://*/*` 主机权限，原因是网页剪存功能需要支持用户在不同网站上保存网页正文。扩展需要在用户访问的网页中检测页面内容、提取正文、处理图片和显示剪存入口。

该权限不会用于无关的数据采集、广告追踪或后台监控。只有在扩展功能运行或用户触发剪存时，页面内容才会被处理。

## 3. 数据收集与使用

### 3.1 本扩展可能处理的数据

为完成网页剪存功能，本扩展可能在用户本地浏览器中处理以下数据：

- 当前网页 URL
- 当前网页标题
- 当前网页正文内容
- 当前网页中的图片、链接和基础结构信息
- 网页可读性解析结果
- 用户选择保存到飞书/Lark 的目标文档或知识库相关信息
- 剪存过程中的状态信息、错误信息和请求结果

当用户执行剪存时，上述被剪存的网页内容会发送到飞书/Lark 官方服务，用于创建或更新用户账号下的云文档、知识库页面或相关附件。

### 3.2 本扩展不会收集的数据

除实现剪存功能所必需的内容外，本扩展不会主动收集以下数据：

- 健康信息
- 财务和付款信息
- 精确地理位置
- 浏览历史记录的持续追踪数据
- 与剪存无关的个人通讯内容
- 与剪存无关的网站账号密码或身份验证凭据
- 用于广告、营销或用户画像的行为数据

### 3.3 诊断和功能日志

本扩展可能向飞书/Lark 服务发送与剪存功能相关的操作状态、错误类型、耗时统计或接口结果，用于判断剪存是否成功、定位上传失败或兼容性问题。这些信息用于扩展功能运行和问题诊断，不用于广告投放或向第三方出售。

## 4. 数据传输

当用户执行剪存操作时，本扩展会将用户选择剪存的网页内容传输到飞书/Lark 官方服务，以保存到用户账号下的云文档或知识库中。根据用户账号所在区域，相关服务域名可能包括飞书、Lark、飞书/Lark API 服务或飞书 CDN 服务。

本扩展不会将用户剪存内容上传到扩展作者自建服务器，也不会将用户数据出售、出租或共享给无关第三方。

## 5. 数据存储

- **本地存储**：用户偏好设置、语言、功能开关、悬浮按钮位置和部分登录状态缓存保存在浏览器本地。
- **云端存储**：用户主动剪存的网页内容会保存到用户授权登录的飞书/Lark 账号下。
- **保存期限**：本地数据保留至用户清除浏览器扩展数据或卸载扩展；云端文档保留期限由用户的飞书/Lark 账号和文档管理设置决定。
- **用户控制**：用户可在浏览器扩展管理页面清除本扩展数据，也可在飞书/Lark 中自行管理、移动或删除已保存的文档。

## 6. 远程代码使用

本扩展的执行代码包含在扩展安装包内，`content_security_policy` 限制扩展页面脚本来源为自身。扩展可能访问飞书/Lark API 和 CDN 服务以完成登录验证、版本信息获取、内容上传和文档创建，但不会从远程地址加载并执行额外代码。

## 7. 第三方服务

本扩展依赖飞书/Lark 官方服务完成登录状态识别、文档创建、内容上传、图片上传和剪存结果打开。用户剪存到云端的内容受飞书/Lark 相关服务条款和隐私政策约束。

除飞书/Lark 官方服务外，本扩展不会主动连接扩展作者自建服务来收集或存储用户数据。

## 8. 用户控制

用户可以通过以下方式管理本扩展的数据和权限：

- 在浏览器扩展管理页面禁用或卸载本扩展
- 在浏览器设置中清除本扩展的本地存储数据
- 在扩展右键菜单或界面中调整语言、自动打开文档、悬浮按钮等功能开关
- 在飞书/Lark 中管理或删除已剪存生成的云文档和知识库内容
- 在浏览器中退出飞书/Lark 登录状态，以停止使用需要账号授权的剪存能力
