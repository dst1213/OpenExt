# 隐私权政策（飞叔网页助手）

最后更新日期：2026-04-25

本隐私权政策适用于浏览器扩展程序「飞叔网页助手」（以下简称"本扩展"）。本扩展用于优化网页浏览体验，包括链接净化、宽屏阅读模式、知乎标题清理等功能。

## 1. 单一用途说明

本扩展的单一用途是：优化用户在各主流网站（知乎、微信公众号、今日头条、CSDN、Bilibili等）的浏览体验，通过清理URL追踪参数、提供宽屏阅读模式、清理页面标题通知数字等功能，提升网页可读性和浏览效率。

## 2. 权限使用说明

### 2.1 activeTab 权限
**请求理由**：用于在用户点击扩展图标或触发快捷键时，获取当前活动标签页的信息（如URL、标题），以便执行链接净化或标题清理操作。此权限仅在用户主动触发扩展时生效，不会在后台持续监控。

### 2.2 storage 权限
**请求理由**：用于在浏览器本地保存用户的偏好设置，包括：
- 知乎宽屏模式开关状态
- 是否显示宽屏切换按钮
- 是否显示操作通知提示
- 其他功能开关状态

这些数据仅存储在本地，不会上传到任何服务器。

### 2.3 scripting 权限
**请求理由**：用于向当前页面注入脚本以执行以下操作：
- 清理知乎页面标题中的通知数字和括号内容
- 显示操作成功/失败的通知提示
- 动态注入宽屏样式（仅在用户开启宽屏模式时）

脚本仅在用户主动触发相关功能时执行，不会在后台自动运行。

### 2.4 tabs 权限
**请求理由**：用于获取浏览器中打开的标签页列表，以便执行"批量清理所有标签页"功能。当用户点击"清理所有标签页"按钮时，扩展会遍历所有标签页，对支持的网站（知乎、今日头条、CSDN、Bilibili）进行URL净化和标题清理。

### 2.5 主机权限（Host Permissions）
**请求理由**：本扩展请求以下网站的主机权限：
- `https://*.zhihu.com/*` - 知乎网站，用于清理URL参数和标题通知
- `https://*.toutiao.com/*` - 今日头条，用于清理URL参数和宽屏阅读
- `https://*.csdn.net/*` - CSDN博客，用于清理URL参数和宽屏阅读
- `https://*.bilibili.com/*` - Bilibili，用于清理URL参数
- `https://mp.weixin.qq.com/*` - 微信公众号，用于宽屏阅读和图片优化
- `<all_urls>` - 用于检测Shift键状态，实现快捷键功能

这些权限仅用于在对应网站上执行链接净化和阅读优化功能，不会用于收集用户数据或追踪用户行为。

## 3. 数据收集与使用

### 3.1 不收集的数据
本扩展**不会**收集以下任何用户数据：
- 个人身份信息（姓名、邮箱、电话等）
- 健康信息
- 财务和付款信息
- 身份验证信息（密码、凭证等）
- 个人通讯内容
- 位置信息
- 用户活动记录（点击、鼠标位置、滚屏、击键等）
- 网站内容（页面文字、图片、视频等）

### 3.2 本地处理的数据
本扩展仅在用户设备本地处理以下数据：
- **当前页面URL**：用于识别是否需要清理URL参数
- **页面标题**：仅用于知乎网站的标题清理功能
- **用户设置偏好**：存储在浏览器本地存储中

所有数据处理均在用户本地完成，不会传输到扩展作者的服务器或任何第三方服务。

### 3.3 远程代码使用
**否**，本扩展未使用任何远程代码。所有代码均包含在扩展程序包中，没有外部脚本引用、动态代码执行（eval）或指向外部文件的模块。

## 4. 数据存储

- **存储位置**：所有用户设置仅存储在浏览器的 `chrome.storage.local` 中
- **存储内容**：仅包括功能开关状态和用户界面偏好设置
- **保存期限**：数据保留到用户手动清除浏览器数据或卸载扩展为止
- **数据共享**：不会与任何第三方共享

## 5. 第三方服务

本扩展**不连接**任何第三方服务或API。所有功能均在浏览器本地实现，不会将任何数据发送到扩展作者的服务器或外部服务。

## 6. 用户控制

用户可以通过以下方式管理本扩展的数据：
- 在浏览器扩展管理页面中清除扩展数据
- 卸载扩展以删除所有本地存储的设置
- 在弹出窗口中随时调整功能开关
